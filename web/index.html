Frequently Asked Questions

How do I install the app?

Download the ZIP from GitHub, move the .app file to your Applications folder. Since the app is currently in beta, you'll need to right-click the app and select 'Open' the first time to bypass the security dialog.

Does it work on Apple Silicon (M1/M2/M3)?

Yes! We distribute a Universal Binary that runs natively on both Apple Silicon and Intel-based Macs for maximum performance.

Is it open source?

Yes, the code is available on GitHub under a Non-Commercial Share-Alike license. You can learn from it, contribute, or build it yourself.
